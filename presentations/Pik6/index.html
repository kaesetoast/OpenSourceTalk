PIK6

HTML5 presentation framework

github.com/SirPepe/Pik6

Features

  • 100% HTML5, CSS and JavaScript (MooTools)
  • Multiple presentation views and presenter mode
  • Programmable slides

Why?

  • Hackable, scriptable presentations
  • Easy embedding of web pages, media, youtube videos â€¦
  • Great performance in modern browsers even with hundreds of slides
  • Presenter mode

Why not?

  • Only works in modern browsers
  • Requires a web server
  • No PDF export

Controls

  • Use the left and right arrow keys to navigate through the slides
  • Move your Mouse to the bottom of the presentation view to reveal more controls
  • The presenter view offers a drop down menu for all slides in the presentation
  • The F5 and ESC keys hide the presentation

Additional keyboard shortcuts

  • Accesskey + r: Reload presentation
  • Accesskey + b: Browser presentations
  • Accesskey + o: "Open URL" dialoge
  • Accesskey + p: Open presenter view

Building presentations

<!-- /presentations/whatever/whatever.html -->
<div id="Pik6-presentation-container">

    <div class="pik6-slide">
        First Slide
    </div>

    <div class="pik6-slide" id="whatever">
        Second Slide
    </div>

</div>

Presenter mode

  • Open presenter.html in a new window to use the presenter mode.
  • The presenter mode offers more control elements, shows a timer and lets you see the currently displayed slide and which slide comes next
  • Currently requires Chrome 8 or newer.

Programming presentations

  • An activate event is fired on the slides when they are shown
  • A deactivate event when hidden
  • A change event is fired on the document object when slides change

Simple programming example

window.addEvent('domready', function(){
    $('Demo').addEvents({
        'activate': function(){
            this.getFirst().set('html', something);
        },
        'deactivate': function(){
            this.getFirst().set('html', '');
        }
    });
});

Scripting without scripts

  • The presentation is just an iframe with a url containing a hash
  • The slide with the id that matches the hash is made visible using the :target selector from CSS3
  • This allows you to trigger visual effects without using JavaScript

Simple CSS "scripting" example

h2#special {
    color: #FFF;
    font-size: 50%;
}
:target h2#special {
    transition: all 1s;
    color: #AE1215;
    font-size: 100%;
}

Slide transitions

  • Use lib/transitions.css in your presentation to enable slide transitions
  • Add a transition class to your slides to use a transition for this slide
  • Example: <div class="pik6-slide pik6-transition-fade">

Transition classes

  • pik6-transition-slide-in-left: slide in from the left
  • pik6-transition-slide-in-right: slide in from the right
  • pik6-transition-slide-in-top: slide in from the top
  • pik6-transition-slide-in-bottom: slide in from the bottom
  • pik6-transition-fade: fade in and out
